职业目标

我的目标是深入探索机器人技术与人工智能，特别是机器学习在自主系统中的应用潜力。我热衷于研发具备环境感知、智能决策与适应能力的机器人系统，推动其在复杂任务中的落地实践。通过多领域的协作与持续的技术积累，我希望为智能系统的可靠性、效率与智能化水平的提升作出贡献。

近期项目

Leo Rover 小组项目

目标：构建一台可自主探索未知区域、识别并分类物体、并在时间和系统限制下安全运行的机器人。
我的贡献：负责软硬件集成，包括 SLAMTEC 激光雷达设置、Intel NUC 配置，以及基于 ROS 的控制开发。
收获：积累了实时机器人系统操作、传感器集成以及搭建完整自主系统的实战经验。

可持续发展中的机器人系统研究

目标：探讨机器人如何在医疗、农业等领域支持联合国可持续发展目标。
我的贡献：分析真实案例并评估隐私、就业流失等伦理问题，推动机器人技术的责任化应用。
收获：加深了对机器人与可持续发展交叉领域的理解，提升了研究与伦理思辨能力。

RPLIDAR A2M12 传感器性能分析

目标：评估 RPLIDAR A2M12 在 Leo Rover 上的导航性能，包括量程、分辨率和表面影响。
我的贡献：使用 ROS2 采集雷达数据，测试不同表面对测距的影响，确保数据一致性与记录完整。
收获：掌握了激光雷达传感、数据分析，以及传感器参数对自主导航效果的影响。

自主移动机器人路径规划算法

目标：通过 GUI 仿真平台实现 A* 路径规划算法。
我的贡献：使用 Python 从零实现 A* 算法，实现基于网格的路径搜索和障碍物规避功能。
收获：深入理解 A* 算法，提升了调试与优化能力，体会到在 GUI 系统中编写高质量代码的重要性。

运算放大器分析、测量与设计

目标：通过理论与仿真深入理解运放工作原理与性能优化电路设计。
我的贡献：完成理论分析、NI Multisim 电路仿真，并设计提升性能的高阶结构。
收获：强化了模拟电路设计与仿真能力，理解了设计选择对电子系统行为的影响。

计算机集群与校园网格项目

目标：构建基于云平台的高性能虚拟集群，研究并优化其并行处理能力与可扩展性。
我的贡献：部署 Azure 虚拟集群，运行 MPI 并行程序并对比不同环境的性能表现。
收获：掌握了云平台搭建、并行计算操作与性能评估分析方法。

天线设计与仿真

目标：设计并仿真偶极子与 Yagi-Uda 天线，通过结构优化提升性能。
我的贡献：优化偶极子天线后，设计出具有定向性的 Yagi-Uda 天线，实现峰值增益 11.1934 dBi。
收获：掌握了天线设计仿真技术，深入理解增益、阻抗等射频性能指标。

电子设计制造与测试课程项目

目标：设计并实现一个 3×3×3 的 LED 立方体，支持无线控制并能响应环境声音进行交互展示。
我的贡献：参与硬件搭建、程序编写，完成麦克风采集音频并通过无线信号控制灯光。
收获：提升了电子设计、嵌入式编程、团队协作与系统调试能力。
